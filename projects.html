Projects

I have worked on quite a number of projects which includes: A calculator for a shoe shop using App inventor. Also, in a group of three, we created a Crime Watch Simulation using a bloom filter, hastable, Object Oriented Programming and Java to model an airport security system. In this project, we intended to simulate an airport security system that which tries to catch criminals as they use the airport by using camera and other surveillance feed to quickly check if a passenger is a documented criminal. However, since the bloom filter can produce false postives, we use a hashtable to store the actual data. This way all positives can be verified by pulling their records. The domain of this project was a github You can have a look! My last project was MOF File Tracking Database System It was a database file tracking system which was designed to track files in the Ministry of Finance so that workers would find it easy to retrieve files and store files effectively.
